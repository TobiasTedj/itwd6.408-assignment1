MY WEB TECHNOLOGIES

My top 5 web technologies that I use:
  • Code Editor: I use Visual Studio Code (VSC) as a text editor. I prefer it due to it's support for a large number of programming languages and the large number of extensions available to aid with software development.
  • CSS Framework: I use Bootsrap. I am still currently learning about Bootstrap but it seems to be a very useful frame
  • Code Hosting: I prefer to use GitHub for code hosting. It is a useful platform for code and project management, issue tracking and collaboration.
  • Cloud Storage: I prefer using Google Drive as an online storage solution. It is straight forward and simple to use, allows for file back ups and sharing and links with all of the other Google services that I use.
  • Web Browser: I use Google Chrome. It is the most popular web browser in the world and using Chrome DevTools for debugging, performance and network analaysis and inspecting HTML and CSS helps with web develompent.
